Aislinn by Daydream Studios

Project Duration: September 2024 - Current
Team: Daydream Studios
Role: Gameplay Programmer
Engine, Languages and Tools: Unreal Engine 5, Blueprints

Aislinn, in which the protagonist shares the same name with, is a hack-slash adventure game that involves traversing dreams with your father's umbrella contained with illustrative floating islands and nightmare phobias attempting to end you journey to finding you father.

Link to itch.io page: https://day-dream-studios.itch.io/aislinn

This game was developed using Unreal Engine 5 and the features of our game has been all implemented using Blueprints.

As a programmer of Aislinn, I have been working on:

  • Player Controls related to combat
  • Player Controls related to traversal
  • Enemy AI behaviour using NavMesh, Behaviour Trees and other AI tools within Unreal Engine 5

Hive Mind by Deathcap Studios

Team: Deathcap Studios
Role: Gameplay Programmer
Engine, Language and Tools: Unity, C#

Hive Mind is an action, top-down rogue-lite in which you explore a light-less, post-apocalyptic world by controlling a swarm of bioluminescent creatures.

Link to itch.io page: https://lw276840.itch.io/hive-mind

Link to GitHub Repository: https://github.com/mydo99/HiveMind.git

This game was developed using Unity and was therefore programmed in C#.

During the development of Hive Mind, I worked on:

  • UI functionality

  • Tutorial
  • Unit stat balancing
  • Load and save system

Diplomatic Emunity by The Emu Agenda

Project Duration: December 2022 - May 2023
Team: The Emu Agenda
Role: Gameplay Programmer, Designer, Pixel Artist, Music Composer
Engine, Language and Tools: Unity, C#

For the team project of the first year of my university course, our team, The Emu Agenda, created a tower defense game based on The Great Emu War in Australia. In Diplomatic Emunity, the player runs around the wastelands of Australia to place down towers whilst fighting back with their gun to defend against a variety of emus.

Link to itch.io page: https://fourman2004.itch.io/politicalemucation

Link to GitHub Repository: https://github.com/mydo99/TheEmuAgenda.git

This game was developed in Unity and was therefore programmed in C#. There was also a few constraints for this project in particular as our team was composed of solely programmers and our game had to be based on an event in history. As a programmer, I worked on:

  • Player movement and shooting
  • Implementing some types of towers
  • AI Boss behaviour
  • UI elements including health bars and pause menu
  • Currency system

I also took the initiative to work on other aspects of the game outside of programming including:
  • Tower Art
  • Player character art
  • Music during the main gameplay
  • Stat changes with tower types and enemies
